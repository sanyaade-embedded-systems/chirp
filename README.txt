CHIRP
-----

An RSS-based, distributed Twitter clone. (server-side, not client-side)


INSTALLATION
------------

Upload index.php, config and tweet to your server. Make sure the script has write access to the "tweet" folder. Edit the config.ini file to contain your username, password and time zone.


USAGE
-----

Call up the index.php page. Write tweets, and they will be added to an RSS feed file. Call up the RSS feed by viewing index.php?format=rss
